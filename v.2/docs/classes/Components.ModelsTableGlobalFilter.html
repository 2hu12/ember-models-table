Components.ModelsTableGlobalFilter Class

Extends Ember.Component
Defined in: addon\components\models-table\global-filter.js:4

Global filter element used within models-table.

Its value is used for all data-items and for each columns propertyName.

Usage example:

{{#models-table data=data columns=columns as |mt|}}
  {{mt.global-filter}}
  {{! .... }}
{{/models-table}}
