Components.ModelsTableColumnsHidden Class

Extends Ember.Component
Defined in: addon\components\models-table\columns-hidden.js:4

Table row used within models-table/table-body.

Component used when all columns are hidden and no data items are shown. Usually it contains a simple message.

Usage example:

{{#models-table data=data columns=columns as |mt|}}
  {{#mt.table as |table|}}
    {{#table.body as |body|}}
      {{#if body.allColumnsAreHidden}}
        {{#body.columns-hidden as |ch|}}
            Any message here
        {{/body.columns-hidden}}
      {{else}}
        {{! ... }}
      {{/if}}
    {{/table.header}}
    {{! ... }}
  {{/mt.table}}
  {{! .... }}
{{/models-table}}
