Components.ModelsTableColumnsDropdown Class

Extends Ember.Component
Defined in: addon\components\models-table\columns-dropdown.js:5

Dropdown with list of table's columns used within models-table.

It allows to toggle visibility for column sets or single column.

Columns that should not be hidden must have property canBeHidden set to false.

Usage example:

{{#models-table data=data columns=columns as |mt|}}
  {{mt.columns-dropdown}}
  {{! .... }}
{{/models-table}}
